General

TransCalc saves projects as a combination of one design verification and multiple circuit refinement calculations. Once a design is opened and the OK BUTTON is pressed, it is temporarily saved under ï¿½Design Verificationï¿½. It can be reopened by double clicking on the name. The design can be modified and resaved.

TransCalc does not allow calculations of multiple designs. If another design is desired after one is already chosen, the user can choose ï¿½Design Verificationï¿½ from the ï¿½Functionsï¿½ menu or the ï¿½Design Verificationï¿½ speed button. The program will display a message box asking whether to delete the current design. If the yes button is pressed, the existing design will be deleted and the design dialog will appear to select another design.

The user may also delete an existing design or refinement by selecting the desired item for deletion and pressing the delete key.

Commands

New Project Clears all previous calculations.

Open Project Opens an existing file.

Save Project Saves all data to an existing file.

Save Project As Saves all data to a file specified by the user.

Print Project Prints a project summary.

Exit Quit TransCalc
