Configuration Program

Associated Application

TransCalc may be configured for either US Customary (default) or SI units with the xconfig.exe configuration program. Select by pressing the appropriate RADIO BUTTON.

The User Notice can be disabled by unchecking the Display User Notice CHECK BOX.

TransCalc must be restarted for these changes to take effect. Data files saved in one system of units cannot be opened when TransCalc is configured for the other system.
