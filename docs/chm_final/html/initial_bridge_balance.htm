Inputs (Limits)

Imbalance (10 to 10 mV/V)

The initial bridge imbalance for an unloaded transducer is entered in mV/V.

Imbalance (50 to 10,000 ohms)

The bridge resistance, measured at P+ and P-, is entered in the EDIT BOX provided.

Resistor Alloy From the list box, choose the balancing resistor material to be used. There is a choice between Constantan (A Alloy), Manganin, and Karma (K Alloy).

Outputs

Zero Balance Resistance

The value of resistor needed to balance the bridge to zero output. The calculation assumes that each bridge arm has approximately equal resistance values.

Bridge Arm

The location for insertion of the zero balance resistor.

Resistors

Once the balance resistor value is computed, the user has a choice of selecting E01 series ladder resistors (for A-Alloy and K-Alloy) or wire (for all 3 resistor materials). The E01 ladder adjustment diagram or wire table can be displayed by selecting the CUT BUTTON.
