Transcalc User Guide

Transcalc is a browser-based engineering environment for designing and compensating strain-gage transducers. It replaces the legacy Delphi Xcalc application with a modern, interactive interface.

Getting Started

Open Transcalc in any modern browser. The home screen shows the Calculator Gallery — a grid of all available transducer types organized by category. Click any card to open that calculator.

To return to the gallery from inside a calculator, click the ← back arrow in the top-left corner next to the calculator name.

Inside a Calculator

Every design calculator follows the same layout:

3D Model

An interactive 3D preview of the transducer geometry. The model updates as you change input parameters. Orbit, zoom, and pan with the mouse. Click 3D Model to collapse or expand this section.

Inputs

A grid of labeled numeric fields. All inputs accept direct typing. Results update immediately — there is no Compute button. Fields that are out of engineering range may be highlighted.

Results

A table of calculated values that updates live as you change any input.

2D Sketch Binocular Beam

The Binocular Beam calculator includes a full 2D engineering sketch (side view + top view) with dimensions, centerlines, gage locations, and load arrows, all driven by the input parameters.

FEA Selected Calculators

Some calculators include a finite-element analysis viewer showing mesh, strain contours, deformed shape, and boundary conditions. Switch between modes using the buttons in the viewer.

Design Calculators

Design calculators compute gage strain, sensitivity (mV/V full-span), and related design parameters for a given transducer geometry and loading. Common outputs shared by all design calculators:

  • Average gage strain (µε) — expected signal strain at the gage location
  • Min / Max gage strain (µε) — strain range across the gage length (indicates gradient error)
  • Full-span sensitivity (mV/V) — bridge output at rated load given the gage factor
  • Natural frequency (Hz) — first resonant mode of the loaded transducer

Bending Beams

Cantilever Beam

A beam fixed at one end with a transverse load at the free end. Gages are bonded to the top and bottom surfaces near the fixed support where bending stress is highest.

Key InputsKey Outputs
Beam length, cross-section width & height, applied force, modulus, Poisson's ratio, gage factor, gage length, gage offset from wall Min/Max/Average strain, strain gradient, natural frequency, full-span sensitivity

Binocular Beam

A rectangular beam with two circular holes aligned along the beam centerline, connected by a narrow slot. The geometry concentrates bending stress in the thin wall above and below the holes, producing high sensitivity for a compact size. Common in precision load cells.

Key InputsKey Outputs
Total beam length, beam height, beam width (depth), hole radius, hole spacing, minimum wall thickness, applied force, gage length, modulus, gage factor Gage strain, full-span sensitivity, natural frequency

Additional features: Full 2D engineering sketch with dimensions, 3D model, FEA mode with strain contour maps.

The minimum wall thickness (t) is the distance from the top of each hole to the reference bending plane. Decreasing t increases sensitivity but reduces stiffness and may cause yielding at lower loads.

Geometry constraint: The slot half-height must be less than the hole radius for a valid binocular shape. If the beam height is too large relative to hole radius and wall thickness, the slot walls cannot connect to the holes.

Dual Bending Beam

Two parallel beams connected at both ends, loaded at the midpoint. Insensitive to off-center loading; common in platform-style load cells.

Reverse Bending Beam

A simply-supported beam (supported at both ends) with a center load. Gages are placed near the supports where bending stress is highest. Bending sign is opposite to a cantilever, hence "reverse."

S-Beam

An S-shaped flexure element for tension/compression load cells. The S-shape creates bending in two sections simultaneously, allowing all four arms of a full bridge to be active.

Column (Axial) Transducers

Column transducers carry axial (compressive or tensile) loads. Gages at 0° are active (along the load axis); gages at 90° are passive (Poisson compensation).

Square Column

Solid square cross-section axial pillar. Inputs: cross-section width, active length, applied force, modulus, Poisson's ratio, gage factor. Outputs: axial strain, transverse strain, full-span sensitivity.

Round Solid Column

Solid circular cross-section axial pillar. Same inputs/outputs as Square Column with a diameter instead of width.

Round Hollow Column

Hollow annular cross-section axial pillar. The bore reduces weight and increases surface strain for a given load. Inputs add inner diameter (bore).

Shear Transducers

Shear transducers measure transverse shear force. Gages at ±45° to the beam axis see the principal strains from shear at their maximum.

Square Shear Beam

Rectangular section shear web. Inputs include beam length, cross-section, applied shear force, gage location, and material properties.

Round Shear Beam

Circular section in transverse shear. Accounts for the non-uniform shear stress distribution across a circular section.

S-Beam Shear

An S-beam used in shear mode. Gage orientation is ±45°.

Torque Transducers

Torque transducers measure torsional moment. Gages at ±45° to the shaft axis see the equal and opposite principal strains from shear stress.

All torque calculators report: shear strain (µε), normal strain (µε), and full-span bridge output (mV/V) for a four-gage ±45° full bridge.

Square Torque

Solid square shaft. The stress distribution in a square section is non-uniform — highest at the midpoint of each side. The calculator applies the appropriate shape correction. Additional outputs include min/max strain and gradient (%) to show variation across the gage length.

Inputs add gage length for gradient computation.

Round Solid Torque

Solid circular shaft. Stress is proportional to radius and uniform around the circumference — the simplest torque case.

Round Hollow Torque

Hollow circular shaft. The bore removes low-stress material from the center, increasing outer surface strain for a given torque.

Pressure Diaphragm

A flat circular diaphragm clamped at its perimeter, loaded by uniform differential pressure. Gages bonded near the center measure tangential strain; gages near the rim measure radial strain.

Key InputsKey Outputs
Applied pressure, diaphragm diameter, diaphragm thickness, modulus, Poisson's ratio Radial strain (µε), tangential strain (µε)

A common full-bridge configuration uses radial gages near the clamped edge and tangential gages near the center to maximize output and reject common-mode effects.

Multi-Axis Force/Torque Sensors

Multi-axis sensors measure multiple components of force and moment simultaneously. They output a sensitivity matrix rather than a single sensitivity value.

6-DOF F/T Cross-Beam

A Maltese-cross flexure with four arms, each instrumented with bending and shear gages. Standard topology for robot wrist and ankle force/torque sensors.

Measures: Fx, Fy, Fz, Mx, My, Mz.

Key InputsKey Outputs
Outer ring radius, inner hub radius, arm width, arm thickness, arm length, material properties Full 6×6 sensitivity matrix, natural frequencies (Fz/Fx/Fy/Mz modes), working bandwidth

Joint Torque Sensor (JTS)

A spoke-flexure disk used in series-elastic actuators and compliant robot joints. Radial spokes carry torsional load by bending.

Measures: Primarily torque; Fx/Fy lateral forces appear as crosstalk.

Key InputsKey Outputs
Number of spokes, outer/inner radius, spoke width, spoke thickness, material properties Torque sensitivity (mV/V per N·mm), lateral force sensitivity, natural frequencies

Hexapod F/T Sensor

A Stewart-platform geometry with six struts in three co-planar pairs. Each strut carries only axial load. The 6×6 Jacobian maps strut forces to the six-component wrench.

Measures: All six DOFs.

Key InputsKey Outputs
Top ring radius, bottom ring radius, strut angle, material properties 6×6 Jacobian, Jacobian inverse (calibration matrix), per-DOF sensitivity, isotropy index

3-Arm F/T Cross-Beam

A three-arm flexure with arms at 120° spacing. More compact than the 4-arm cross-beam; uses six gages (two per arm: one bending, one shear). Decodes all six DOFs via a 6×6 sensitivity matrix.

Measures: Fx, Fy, Fz, Mx, My, Mz.

Compensation Calculators

After designing a transducer, compensation calculates resistor values to correct for bridge unbalance, temperature drift of zero output, and temperature variation of span (sensitivity).

Zero vs Temperature

Determines the resistance of a temperature-sensitive wire (e.g., Balco or Nickel) to cancel zero-shift as temperature changes.

Inputs: Zero output at two or three temperatures, bridge resistance, wire material TC.
Outputs: Required compensation wire resistance and length.

Zero Balance

Computes a shunt or series resistor to null a fixed bridge unbalance (zero offset at room temperature).

Inputs: Bridge unbalance (mV/V or µε equivalent), bridge resistance.
Outputs: Null resistor value.

Span Temp 2-Point

Linear temperature correction for span using measurements at two temperatures.

Inputs: Span at two temperatures, material modulus–temperature relationship.
Outputs: Compensation element value.

Span Temp 3-Point

Quadratic temperature correction for span using measurements at three temperatures.

Inputs: Span at three temperatures.
Outputs: Polynomial correction coefficients, compensation element value.

Shunt Optimization

Finds the optimal shunt resistor to realize a target span shift for a trim network ladder.

Inputs: Bridge resistance, target span shift (%), available shunt range.
Outputs: Optimal shunt value, achieved span, trim topology recommendation.

Span Set

Calculates a series or shunt resistor to scale output to a target full-span value after calibration.

Inputs: Measured span, target span, bridge resistance.
Outputs: Span-set resistor value.

Simulated Span

Computes a simulation circuit for verifying calibration without applying mechanical load. A precision resistor in one bridge arm simulates a known strain.

Inputs: Bridge resistance, target simulated output.
Outputs: Simulation resistor value.

Trim Visualizer

The Trim Visualizer displays a ladder resistor network used for fine-tuning (trimming) compensation resistors. Ladder networks have a series of parallel "rungs" that can be cut or left intact to achieve a target resistance.

The visualizer shows:

  • The ladder topology (series rails with parallel rungs)
  • Which rungs are cut (open) vs. intact (closed)
  • The resulting equivalent resistance

Use this alongside the compensation calculators to translate a target resistance value into a specific rung-cut pattern for the physical trim network.

Project Save and Load

Transcalc saves your work to a .tcalc.json file and reloads it later. No account or server connection is required.

Saving

Click Project in the top bar. A panel opens showing current metadata. Click Save to download a .tcalc.json file.

The saved file stores: the currently selected calculator, the unit system (SI or US), and input values for each calculator you have visited during the session.

Loading

Click Project → Load, then select a .tcalc.json file from disk. The app restores the unit system, navigates to the saved calculator, and restores all inputs.

Printing a Report

Click Print Report in the top bar. The browser's print dialog opens. The page is formatted to include the 3D model, inputs, and results table, suitable for an engineering work record.

For best results, use landscape orientation and set margins to minimum.

Help System

Click Help in the top bar to open the help panel. The panel shows a two-column view:

  • Left: A searchable list of all calculator topics
  • Right: The detailed help page for the selected topic (ported from the original Delphi CHM help system)

Type in the search box to filter topics by name or category. Each help page includes formula descriptions, variable definitions, application notes, and wiring diagrams.

The Multi-Axis calculators (6-DOF F/T, Hexapod, JTS, 3-Arm) do not yet have legacy help pages. Refer to this guide for those calculators.

Quick Reference

Gage Color Coding

All 2D sketches and 3D previews use a consistent color scheme for strain gage locations:

ColorRole
OrangeActive gage — positive strain (+ε)
BlueActive gage — negative strain (−ε)
GrayPassive (dummy) gage — temperature compensation only, no signal strain

In a full Wheatstone bridge, two orange and two blue gages (or orange + gray) are wired in opposing arms to maximize output and reject temperature effects.

InputMeaning

Modulus (E)Young's modulus of the spring element material
Poisson's ratio (ν)Lateral strain ratio; typically 0.28–0.33 for steel, 0.33 for aluminum
Gage factor (GF)Resistance sensitivity of the strain gage; typically 2.0–2.15 for foil gages
Gage lengthPhysical length of the gage grid along the strain axis
Gage offsetDistance from a reference point (fixed wall, support) to the gage center

Typical Workflow

  1. Select a transducer type from the gallery based on how the sensor will be loaded (bending, axial, shear, torque, pressure, or multi-axis).
  2. Enter geometry and material parameters. Start with approximate dimensions. Use the 3D model to visually confirm the geometry.
  3. Review the results. Check that gage strain is in an appropriate range (typically 500–2000 µε at full scale for foil gages). Check that full-span sensitivity is acceptable (typically 1–3 mV/V at 5–10 V excitation).
  4. Iterate dimensions to optimize sensitivity, stiffness, natural frequency, or safety factor against yield.
  5. Run a compensation calculator if temperature compensation is required. Start with Zero vs Temperature to correct zero drift, then apply a Span Temperature correction if span variation is significant.
  6. Use the Trim Visualizer to convert the compensation resistor value into a specific ladder-network rung-cut pattern.
  7. Save the project as a .tcalc.json file for records.
  8. Print a report for documentation.
